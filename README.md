# CoachCat Zone Summary

A Chrome extension that adds a weekly power zone distribution panel to [CoachCat](https://app.fascat.ai).

![Zone Distribution Panel](https://img.shields.io/badge/version-1.4-green)

## Features

- Displays time spent in each power zone for the current week
- Shows zone ranges based on your FTP
- Floating panel with toggle button for easy access

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked**
5. Select the folder containing this extension

## Usage

1. Log in to [CoachCat](https://app.fascat.ai)
2. Click the 📊 button in the top right corner to view your weekly zone distribution
3. Click the × or the toggle button again to hide the panel

## Requirements

- Google Chrome browser
- Active CoachCat account with training data
